## Platform and Development Environment

- **Operating System:**  Windows 10 / 11

- **Language:**  C# 
- **Framework**  .NET Framework (Windows Forms) 

---

## How to Run

### Prerequisites
- [Visual Studio Community 2022](https://visualstudio.microsoft.com/vs/community/) with **.NET desktop development** 

### Steps

1. Download the repository .zip file and extract it
2. Open Visual Studio Community 2022
3. Select **File → Open → Project/Solution**
4. Navigate to `FavoriteBooks.csproj` and open it
5. Press the green **Start** button at the top (or press `F5`) to build and run

## Notes

- This project uses Windows Forms and **must** be opened in Visual Studio Community 2022. It is **not** supported in VS Code or the C# Dev Kit extension.
- Data persists between runs via the `.txt` files in the project directory
- Payment processing is simulated, no real transactions are made
